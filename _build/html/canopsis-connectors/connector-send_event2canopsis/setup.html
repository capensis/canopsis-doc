connector-send_event2canopsis is a script that execute a powershell script and publish an event given as argument to Canopsis/REST API

Setup¶

Requirements¶

Requirements :

connector-send_event2canopsis is based on Powershell and can be executed by powershell v2.x and >= 3.x Pay attention, 2 scripts exist depending on powershell’s version.

connector-send_event2canopsis for powershell >= 3.x used cmdlet : * ConvertTo-JSON, * ConvertFrom-Json, * Invoke-WebRequest.

To install the connector, just copy the ps1 file into a directory on your windows machine.
